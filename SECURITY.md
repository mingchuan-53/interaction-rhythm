# 安全与隐私

交互节律处理的是个人使用节律数据，因此安全边界比功能数量更重要。

## 当前数据范围

应用只记录：

- 键盘响应次数
- 鼠标响应次数
- 前台应用名称、路径和窗口标题
- 前台应用停留时间
- 键盘 / 鼠标响应对应的前台应用

应用不记录：

- 具体按键内容
- 输入文本
- 鼠标坐标
- 截图
- 浏览器正文内容

## 数据位置

运行数据保存在本机 `data/` 目录或用户配置目录中。仓库 `.gitignore` 默认排除数据库、构建产物和调试截图。

## 报告问题

如果发现隐私或安全问题，请在 GitHub Issue 中描述：

- 复现步骤
- 影响范围
- 你期望的行为

如果问题可能暴露个人数据，请先不要公开粘贴真实数据。
