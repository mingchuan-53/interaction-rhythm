# 交互节律 v1.6

这是交互节律面向普通用户分发准备的一版。重点不是增加数据能力，而是让下载、更新和安装更稳定。

## 主要变化

- 启动后会轻量检查更新，有新版时显示小提示，不会自动安装。
- 设置页增加“自动检查更新”开关。
- 默认更新通道指向 GitHub Release，减少手工配置遗漏。
- 打包脚本同时生成 `交互节律.zip` 和 `interaction-rhythm.zip`。
- 打包脚本会在替换当前版本前停止运行中的旧版本，避免文件占用导致打包失败。
- 新增 Inno Setup 安装器脚本和 `build-installer.ps1`。
- 安装器支持开始菜单、桌面快捷方式和卸载入口，采用每用户安装，不要求管理员权限。

## 下载

普通用户优先下载安装器：

```text
interaction-rhythm-setup-v1.6.exe
```

也可以继续下载便携包：

```text
interaction-rhythm.zip
```

## 隐私说明

本版本仍然只做本地记录，不上传数据。键盘只记录次数，不记录具体按键或输入内容。

## 更新验证

2026-06-12 已完成端到端更新验证：

- 复制 `dist/releases/InteractionRhythm-v1.5` 到独立测试目录。
- 启动测试副本并读取 GitHub `latest` 更新清单。
- `/api/update/check` 返回 `current=1.5`、`latest=1.6`、`has_update=true`。
- 调用 `/api/update/install` 后，测试副本下载 GitHub `interaction-rhythm.zip` 并执行自我替换。
- 更新脚本日志显示 `update started` 和 `update finished`。
- 重启后的测试副本返回 `current=1.6`、`latest=1.6`、`has_update=false`。

结论：`v1.5` 可以发现并完成升级到 `v1.6`。
