# 交互节律 v1.7.1

这一版重点修复更新和设置相关的阻塞感，并强化升级时的数据保留。

## 主要变化

- 检查更新先读取本地缓存状态，再在后台静默确认远程版本。
- 设置页打开时立即显示当前状态，不再等待设置读取完成。
- 更新安装前会备份用户数据，更新包不会覆盖 `tracker.db`、设置和本地图标缓存。
- 更新退出前会执行数据库 checkpoint，减少升级后显示重新计数的风险。
- 节律助手建议减少到三条，并按建议类型去重。
- 节律助手不再把交互节律自身作为分析建议对象。
- 弹窗增加轻量打开和关闭动画。
- 修正升级后本地更新缓存短暂显示旧版本号的问题。

## 下载

普通用户优先下载安装器：

```text
interaction-rhythm-setup-v1.7.1.exe
```

便携包：

```text
interaction-rhythm.zip
```
