# 交互节律 v1.5

这是交互节律开源后的第一版桌面发布。

## 主要变化

- 新增单应用键盘、鼠标响应归属。
- 节律助手改为“结论 / 重点发现 / 建议”，不再展开完整报告。
- 单应用分析增加响应次数和响应密度。
- JSON / CSV 导出增加应用交互记录。
- 补齐 README、开发日志、许可和贡献说明。

## 隐私说明

本版本仍然只做本地记录，不上传数据。键盘只记录次数，不记录具体按键或输入内容。

## 安装

GitHub 发布页下载 `interaction-rhythm.zip`，解压后运行 `InteractionRhythm.exe`。本地构建目录仍保留 `交互节律.zip` 作为朋友测试包名称；应用内检查更新使用稳定的 GitHub 下载链接。
