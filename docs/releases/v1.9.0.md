# 扣舷 v1.9.0

这一版是稳定体验版：产品显示名从“叩舷”调整为“扣舷”，首页围绕“这一天的数字手感”重新收束，并把应用、热力图、回放和图标链路统一起来。

## 主要变化

- 产品显示名统一为“扣舷”，继续保留 `InteractionRhythm.exe`、`interaction-rhythm.zip` 和旧数据路径兼容。
- 首页增加日期看板，可查看前一天、后一天并回到今天。
- 今日摘要升级为“今天的形状”，优先呈现主场应用、键鼠偏向和峰值小时。
- 热力图改为一天一列的小时视图，并增加月视图。
- 点击小时格可查看这一小时的应用构成。
- “应用强度榜”改为“应用手感构成”，排序优先看键鼠响应和手感密度。
- 手感回放默认讲选中日期，最近 7 天回放保留为次级入口。
- 应用图标提取增加失败负缓存，减少坏路径反复提取。
- 新图标母版接入应用图标、托盘图标、网页图标、标题栏图标和发布包资源。
- 发布预检扩展到 release staging 目录，防止数据库、日志、图标缓存和调试截图进入可分享目录或 zip。

## 数据继承

升级继续保留：

- `data/tracker.db`
- `data/settings.json`
- `data/icons/`
- `data/updates/`
- 旧名称图标文件：`交互节律.ico`、`叩舷.ico`

更新包只替换程序文件和必要资源，不覆盖用户数据库和设置。

## 验证重点

- 首页标题显示“扣舷”，副标题显示“看见你的数字手感”。
- 日期切换、今天按钮、小时热力图、月视图、小时详情、应用手感构成和手感回放可用。
- 托盘、标题栏、网页、安装器和发布包图标一致。
- 发布包不包含数据库、日志、缓存、旧更新暂存和调试截图。
- 旧版本应用内检查更新能发现 `1.9.0`。
