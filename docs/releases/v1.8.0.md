# 叩舷 v1.8.0

这一版是叩舷的大更新：产品从“交互节律”正式收束到“看见你的数字手感”，视觉改为更轻、更白、更像个人习惯统计应用的手感仪表盘。

## 主要变化

- 全新浅色仪表盘，使用白色轻卡片、柔和阴影、玫红强度视觉和胶囊控件。
- 默认主题色改为传统绿色，并在设置中加入主题色与中英文切换。
- 首页改为单页仪表盘，应用强度榜扩展为十项双列排行，不再需要滚动查看。
- 启动时会自动扫描并合并旧安装目录、旧 release 和 archive 里的 `tracker.db`，兼容远古版本数据继承。
- 首页聚焦今日手感、手感热力图、应用强度榜和手感回放。
- “节律助手”改为“手感回放”，输出主场应用、敲击冲刺、碎片回返、夜航时段等趣味线索。
- 单应用详情使用“手感响应”“手感密度”“回返”等标签，减少效率评分口吻。
- 产品显示名改为“叩舷”，但继续保留 `InteractionRhythm.exe`、`interaction-rhythm.zip` 和原数据目录，保证旧版本升级和旧数据继承。

## 数据继承

升级不会重建数据库，也不会移动用户数据。旧版的：

- `data/tracker.db`
- `data/settings.json`
- `data/icons/`
- `data/updates/`

都会继续保留。更新脚本仍会先备份 `data`，再替换程序文件，并在启动新版后恢复用户记录。

## 验证重点

- 从旧版升级后，今日数据和应用强度榜仍能读取旧 `tracker.db`。
- 首页标题显示“叩舷”，副标题显示“看见你的数字手感”。
- 手感热力图、应用强度榜、手感回放、设置、导出和检查更新可用。
- 发布包不包含数据库、日志、缓存和调试截图。
