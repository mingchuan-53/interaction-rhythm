# 交互节律 PWA 测试指南

## 快速测试步骤

### 1. 启动应用
```bash
cd D:\10_Projects\10_Products\interaction-rhythm
python main.py
```

### 2. 在浏览器中测试 PWA

#### Chrome/Edge 测试：
1. 打开 `http://localhost:18923`
2. 按 F12 打开开发者工具
3. 切换到 "Application" 标签
4. 检查以下内容：
   - ✅ Manifest 是否加载
   - ✅ Service Worker 是否注册
   - ✅ Cache Storage 是否有缓存

#### 安装 PWA：
1. 地址栏右侧会出现安装图标
2. 点击安装到桌面
3. 或使用安装提示横幅

### 3. 手机端测试

#### 方法一：局域网访问
1. 确保手机和电脑在同一网络
2. 查看电脑 IP：`ipconfig`
3. 手机浏览器访问：`http://[电脑IP]:18923`

#### 方法二：使用 ngrok（公网访问）
```bash
# 安装 ngrok
winget install ngrok

# 启动隧道
ngrok http 18923
```

### 4. 移动端功能检查清单

- [ ] 响应式布局正常
- [ ] 热力图可滑动查看
- [ ] 应用列表可滚动
- [ ] 弹窗从底部弹出
- [ ] 下拉刷新可用
- [ ] 离线提示显示
- [ ] 安装提示显示
- [ ] 添加到主屏幕正常

### 5. PWA 安装测试

#### Android Chrome：
1. 访问应用
2. 点击 "添加到主屏幕"
3. 从主屏幕启动应用
4. 检查是否全屏运行

#### iOS Safari：
1. 访问应用
2. 点击分享按钮
3. 选择 "添加到主屏幕"
4. 从主屏幕启动应用

### 6. 离线功能测试

1. 正常加载应用
2. 断开网络
3. 刷新页面
4. 应该显示离线提示
5. 缓存的数据应该仍然可用

## 常见问题

### Q: 安装提示不显示？
A: 检查以下条件：
- 必须使用 HTTPS 或 localhost
- manifest.json 必须正确配置
- Service Worker 必须注册成功
- 不能已经在独立窗口运行

### Q: Service Worker 不工作？
A: 检查：
- 浏览器控制台是否有错误
- sw.js 文件是否可访问
- 是否使用 HTTPS 或 localhost

### Q: 手机端布局问题？
A: 检查：
- viewport meta 标签是否正确
- CSS 媒体查询是否生效
- 安全区域是否正确处理
